# 🚀 BarakattSpace Webstore (BKWebstore)

A full-stack e-commerce web application designed for international users.  
Built to support product browsing, secure checkout, and complete admin management.

---

## 📌 Project Overview

BarakattSpace is a software development company providing modern software solutions.  
This project is an online webstore system designed to:

- Sell products online
- Provide a smooth user experience
- Build a global brand presence

---

## ✨ Features

### 👤 User Features
- User registration & login
- Browse products by categories
- Search and filter products
- View product details
- Add to cart & manage cart
- Secure checkout system
- Order tracking & history

### 🛠️ Admin Features
- Add, update, delete products
- Manage product categories
- View and manage orders
- Track sales

---

## 🧱 System Design

### 📊 Core Entities
- User
- Product
- ProductCategory
- Order
- Sale
- Feedback

### 🔗 Relationships
- A user places orders  
- An order contains multiple products  
- A product belongs to a category  
- A user can leave feedback on products
